# 增强过程

## 1. 分析原始提示词
- 识别模糊或不清楚的部分
- 确定缺失的上下文或背景信息
- 评估当前的具体性和可操作性
- 识别可能的改进领域

## 2. 应用增强策略
- 添加具体的细节和要求
- 包含相关的背景信息
- 改进结构和组织
- 澄清期望和输出要求
- 添加质量标准和约束

## 3. 验证增强效果
- 确保增强后的提示词更清晰
- 验证所有必要信息都已包含
- 检查提示词是否更具可操作性
- 确认保持了原始意图
